# W3C CCG Archiver

This repository is the configuration for the W3C Credentials Community Group (CCG) instance of the [cg-archiver](https://github.com/w3c-ccg/cg-archiver) tool.

It automatically archives Google Meet recordings and transcripts for CCG  meetings to community-owned infrastructure at
[meet.w3c-ccg.org](https://meet.w3c-ccg.org/archives/).

## What It Archives

The following recurring meetings are configured for archival:

- **CCG Atlantic Weekly** — Main weekly meeting of the W3C CCG
- **VCs for Education** — Verifiable Credentials for Education task force
- **CCG VCALM** — VC Application Lifecycle Management task force
- **CCG Data Integrity** — Data Integrity specification work
- **CCG Incubation** — Incubation calls for new work items
- **CCG Asia-Pacific** — Asia-Pacific regional meeting
- **DID WG Special Topic** — DID Working Group special topic calls
- **VCWG Spec Refinement** — VC Working Group specification refinement
